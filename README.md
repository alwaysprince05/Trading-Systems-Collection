## Created by: Prince Maurya

## 📈 Overview

This repository features three high-performance quantitative trading systems, implementing core market strategies: **Trend Following**, **Momentum Portfolio**, and **Mean Reversion**. Designed for modern traders, these systems provide visual clarity and actionable insights through high-fidelity dashboards.

---

## 🛠 Featured Trading Systems

### 1️⃣ Trend Following System (MACD + EMA + ATR)
Captures sustained market moves using dual-layer confirmation.
* **EMA Crossover**: Identifies trend transitions.
* **MACD Confirmation**: Validates momentum strength.
* **ATR Risk Management**: Dynamic stop-losses based on volatility.

### 2️⃣ Multi-Asset Momentum Portfolio
A sector-rotation engine for risk-adjusted growth.
* **Dynamic Rotation**: Shifts capital to top-performing sectors.
* **Volatility Adjustment**: Balances risk across diverse assets.
* **Performance Analysis**: Equity curves and correlation heatmaps.

### 3️⃣ Mean Reversion System (RSI + Bollinger Bands)
Statistical arbitrage targeting price deviations from the mean.
* **RSI Indicators**: Detects overbought/oversold exhaustion.
* **Bollinger Bands**: Visualizes price stretch and mean-reverting targets.
* **Visual Dashboards**: Multi-panel performance tracking.

## 🚀 Getting Started

### 🖥 Launching the Web Dashboard
I have enhanced this project with a **Streamlit Web Dashboard** to visualize all systems directly in your browser.

1. **Install Dependencies**:
   ```sh
   pip install streamlit pandas numpy matplotlib yfinance
   ```

2. **Run the Dashboard**:
   ```sh
   streamlit run app.py
   ```
